Portal v1.0 - CF API, DB API, OBJECT STORAGE API 통합 버전

## Portal API
Portal API 란? CF API, DB API, OBJECT STORAGE API 통합적으로 제공하는 서비스이다.
현재는 각각의 서비스로 분리하는 작업을 진행하고 있다.
- v1.0 - FINAL까지는 통합으로 제공
- v2.0 - 각각의 API 서비스로 제공
